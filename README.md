# 🤖 PolyBot: Polymarket 赔率自动监控助手

这是一个部署在 **GitHub Actions** 上的自动化脚本，专门为 **M4 Mac mini** 环境开发，旨在每日定时汇总 Polymarket 上的热门事件赔率并发送至指定邮箱。

## 🌟 功能特点
* **自动化运行**：完全托管在 GitHub 云端，无需本地 24 小时开机。
* **定时推送**：每天北京时间 **08:00** 和 **12:30** 准时发送最新赔率日报。
* **精简高效**：基于 Python 编写，仅需 `requests` 库即可完成 API 抓取。
* **隐私保护**：通过 GitHub Secrets 加密存储邮箱授权码，确保账号安全。

## 🛠️ 技术栈
* **语言**: Python 3.9+
* **运行环境**: GitHub Actions (Ubuntu Latest)
* **数据源**: Polymarket API
* **通知方式**: SMTP 邮件推送 (QQ Mail)

## 📅 定时任务说明 (Workflow)
项目配置了双时段自动触发：
1.  **晨报**：`00:00 UTC` (北京时间 08:00)
2.  **午报**：`04:30 UTC` (北京时间 12:30)

## 🚀 部署记录 (Mac mini M4)
1.  本地开发并测试 `main.py`。
2.  配置 `.github/workflows/daily.yml` 实现云端调度。
3.  在 GitHub 仓库中设置 `MAIL_PASS` 环境变量。
4.  使用 Git 推送至远程仓库触发自动构建。
